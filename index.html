Oracle PL/SQL Payroll System Overview

About the System: This database frontend directly integrates with the PL/SQL schemas. Standard calculations for allowances (HRA & Medical) and attendance deductions (Unpaid Leaves, Absences, Half-days, Late check-ins) are computed using actual PL/SQL packages ported to the dashboard simulator. You can inspect tables or run SQL commands directly inside the Database Console tab.

Work Policy & Deductions

  • Basic Allowance: 15% added to base pay (10% HRA + 5% Medical).
  • Late Grace Period: 3 grace late check-ins allowed. Each day past 3 results in a 20% daily wage penalty.
  • Absence / Unpaid Leave: 100% daily wage deducted.
  • Half-Day check-in: 50% daily wage deducted.
